Gaming Quiz API

About

GamingQuizApi is a free API that puts your knowledge to the test with a collection of video game-related questions.

Usage

Retrieve a list of cards.

GET https://gamingquizapi.azurewebsites.net/api/card

Retrieve a single card with an id.

GET https://gamingquizapi.azurewebsites.net/api/card/{id}

Retrieve a random card.

GET https://gamingquizapi.azurewebsites.net/api/random/card

Schema

{

  "id": 0,

  "question": "string",

  "answer": "string"

}
